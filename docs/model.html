#配置模型

在 Ract 中可以通过设置 - 模型管理 来配置和管理 AI 模型。

#添加通用模型

通用模型适用于大多数聊天、图像（视频）生成任务，目前 Ract 提供了 3 种不同的方式来添加模型。

#一键授权

目前 Ract 对 阿里云 - 百炼、火山引擎 - 方舟、OpenRouter、DeepSeek 等主流模型提供商提供了一键授权功能，只需点击对应的授权按钮，即可打开内置浏览器，加载模型提供商的登录页面。

在登录完成后，点击顶部的 点此获取 API key 按钮，将页面中展示的 API Key 填写入右侧的输入框中，即可完成授权。

后续通过一键授权添加模型，则无需再输入 API Key，会自动填入。

#选择添加模型

Ract 集成了 models.dev 提供的模型列表，用户可以通过点击 选择添加模型 按钮，在弹出的模型列表中选择需要添加的模型。

通过这种方式添加模型，需要前往 模型提供商 处申请 API Key，并在添加模型时填写。

#添加自定义模型

通过手动输入 AI 的 API 地址和访问 API 的 key、模型名称来添加模型。

通过这种方式添加模型，需要前往 模型提供商 处申请 API Key，并寻找 模型提供商的 API 地址，操作虽然复杂一点但更灵活，也可以通过 JSON 格式添加一些自定义的模型参数。
