# 快速开始

## 项目介绍
Ract 是完全免费的基于 Swift 开发的自动化 AI Agent 软件，通过较小的包体积提供了自动化动作、智能记忆、MCP、Skills、ACP、SubAgent 和其他高级特性，帮助用户轻松实现各种基于 AI 的自动化任务。

## 安装
点击 [https://ractai.com/dist/ract-latest.dmg](https://ractai.com/dist/ract-latest.dmg) 免费下载最新版本的 Ract 安装包，然后双击安装包进行安装。

## 功能介绍

1. 自动化动作：把发送给 ai 的“多轮消息”保存为自动化动作，可以自动定时调用。比如每天早上自动调用麦当劳 mcp 帮我领优惠券，然后发系统消息通知

2. 支持加载 skills 技能，支持输入 SKILL. md 链接，一键下载安装

3. MCP、ACP 协议支持

4. 网站连接：用 `已登录态` 让内置工具去访问一些网站获取内容（登录态也保存在本地）

5. 语音朗读：直接集成苹果系统的TTS能力，开箱即用

6. 多种内置工具：网络搜索、内容获取、截图、命令执行等

7. SubAgent 任务委派：让专业的智能体完成专业的任务，并且可以减少主任务的 token 消耗

8. 会话文件：自动展示会话文件目录，可以直接预览 ai 生成的 html 页面、图片等等

9. 任务拆分与管理，可以让 Agent 按照多步骤更好地完成复杂任务（类似 cursor 的 todo_write 工具）

10. 记忆：自动记忆更新，也可以手动添加记忆，在聊天的过程中自动选择调用记忆，ai 也会自动更新一些记忆

11. 支持很多 AI 提供商 `一键接入`

12. 流式输出、Markdown 渲染

13. 深度思考
