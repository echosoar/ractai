# 会话网关

会话网关提供了将 Ract 的能力暴露给其他应用的能力，类似于 OpenClaw，可以在飞书等第三方应用中集成 Ract。

## 飞书集成

1. 进入到飞书开放平台（https://open.feishu.cn/app），创建一个新的应用。
![](/doc-static/session/feishu-app-select.png)

2. 填写一些基本信息
![](/doc-static/session/feishu-app-create.png)

3. 授予权限

![](/doc-static/session/feishu-grunt.png)

4. 发布应用

5. 在飞书开放平台，点击 `基础信息` - `凭证与基础信息`，复制应用的 `App ID` 和 `App Secret`，并在 Ract 的 `设置` - `会话网关` 中进行配置。

![](/doc-static/session/feishu-add.png)

6. 在飞书中创建群聊，搜索添加机器人，输入机器人名称，选择刚才创建的应用，完成添加。

7. @这个机器人，发送消息进行对话。


## 权限

### 网关权限

* 处理消息：允许会话网关处理来自第三方应用的消息，并将其转发给 Ract 进行处理。
* 清空会话：清空历史消息，在会话中发送 `/清空会话` 可以清空当前会话的历史消息记录。
* 清空任务队列：由于会话网关会将接收到的消息转换成任务放入 Ract 的任务队列中挨个执行，在群聊中如果任务堆积过多造成执行缓慢，可以发送 `/清理任务队列`，让会话网关清空这些任务队列。

### 工具权限

会话网关在处理消息时，可能会调用一些工具来完成任务，例如网络搜索、内容获取、文件操作等。

### 目录授权

默认会话网关只能访问当前的会话目录，如果需要访问其他目录，可以在 `设置` - `会话网关` 中 `授权更多目录访问`。

> 授权更多目录需要谨慎操作，由于 AI 可以操作这个目录中的文件，比如查看、编辑和删除等，所以需要谨慎，避免造成不必要的损失。
