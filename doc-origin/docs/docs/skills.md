# 技能 Skills

## 介绍

技能（Skills）是一种可以扩展 Ract AI 能力的插件机制。每个技能通过一个 `SKILL.md` 文件描述，其中包含该技能的功能介绍、使用说明和相关工具定义，AI 在加载技能后可以根据技能描述来使用对应的能力。

## 安装技能

在 `设置` - `技能 Skills` 中，可以添管理技能。

Ract 提供了一个默认的技能目录，可以一键打开该目录来添加技能。技能的安装非常简单，只需要将包含 `SKILL.md` 文件的技能文件夹放入该目录中即可。

也可以通过 Ract 提供的 `下载安装技能` 功能，输入技能的 URL 地址来安装技能。

![](/doc-static/usage/down-skill.png)

## 使用技能

技能安装后，AI 在处理相关任务时会自动通过内置的「Skills」工具加载并使用对应的技能描述。

AI 会根据当前任务的需要决定是否调用某个技能。可以在聊天窗口的「工具」选择中开启或关闭 Skills 工具。

![](/doc-static/usage/skill-use-tool.png)

在 `设置` - `内置工具` - `Skills` 中，可以配置加载技能时是否需要用户二次确认。开启「无需二次确认」选项后，AI 可以直接加载技能而无需用户确认。
