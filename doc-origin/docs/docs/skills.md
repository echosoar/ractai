# 技能

## 介绍

技能（Skills）是一种可以扩展 Ract AI 能力的插件机制。每个技能通过一个 `SKILL.md` 文件描述，其中包含该技能的功能介绍、使用说明和相关工具定义，AI 在加载技能后可以根据技能描述来使用对应的能力。

## 安装技能

在 `设置` - `技能` 中，可以添加和管理技能。

点击 `添加技能` 按钮，然后输入技能的 `SKILL.md` 文件链接（URL），Ract 会自动下载并安装该技能。

## 使用技能

技能安装后，AI 在处理相关任务时会自动通过内置的「Skills」工具加载并使用对应的技能描述。

在 `设置` - `内置工具` - `Skills` 中，可以配置加载技能时是否需要用户二次确认。开启「无需二次确认」选项后，AI 可以直接加载技能而无需用户确认。

## 技能说明

技能的加载通过 Ract 的内置「Skills」工具实现，AI 会根据当前任务的需要决定是否调用某个技能。可以在聊天窗口的「工具」选择中开启或关闭 Skills 工具。
