# 聊天会话

## 界面交互

![](/doc-static/usage/chat.png)

1. 选择附件：选择图片、目录或者拍照等
2. 选择模型：可以选择已经添加的模型进行对话，也可以在聊天过程中随时切换模型
3. 工具：选择开启或关闭本次会话是否使用一些内置工具，例如网络搜索、内容获取、截图、命令执行等，以及管理 MCP 工具
4. 更多：包含了本地会话的一些信息，如当前会话目录、会话token消耗、会话消息延时、清除会话消息等
5. 消息发送：可以点击此按钮发送消息，也可以使用快捷键 `Enter` 来发送消息
6. 语音朗读：对于包含文本的消息，可以点击此按钮让系统朗读消息内容
7. 消息折叠：消息太长可以点此按钮进行折叠，点击展开可以查看完整消息内容
8. 智能体：选择使用一些内置或自定义的智能体进行对话
9. 搜索：通过关键词搜索会话消息
10. 新的聊天：新开启一个聊天会话
11. 编辑之前的消息，可以重新以编辑的消息重新生成
12. 重新生成：从选择重新生成的消息开始，重新生成后续的消息内容

## 聊天配置

在聊天消息上点击右键，选择 `配置聊天` 按钮，则可以对聊天的一些参数进行配置。

![](/doc-static/usage/chat-config.png)

上下文数量：聊天时会携带之前的消息作为上下文发送给模型，设置上下文数量可以控制携带`多少条`之前的消息。上下文过多则会使 Token 消耗大大增加。

匹配度阈值：聊天时会根据当前的消息和记忆库中的记忆进行匹配，根据匹配程度选择前 n 条记忆作为上下文发送给模型
