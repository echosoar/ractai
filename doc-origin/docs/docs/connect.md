# 网站连接

## 介绍

可以将一些需要登录后才能访问数据的网站，添加到`网站连接`中，点击进行连接并在对应的网站登录后，Ract 会在`您电脑的本地` 保存该网站的登录态（Cookies、LocalStorage 等信息），之后在使用内置工具（网络搜索、网页内容获取等）访问该网站时，就会携带该登录态进行访问，从而获取到登录后的内容。

![](/doc-static/usage/connect.png)

## 使用

可以在 `设置` - `网站连接` 中进行配置，之后点击 `进行连接` 按钮，Ract 会自动打开一个浏览器窗口，进入该网站的登录页面，您可以在该页面进行登录操作。

登录成功后，再点击 `完成连接` 按钮，Ract 会自动保存该网站的登录态信息。
